Deterministic Code Generation

The headline capability: project one Prisma data-model contract into a working all-Python app (Pydantic + SQLModel + FastAPI + HTMX) at $0 LLM cost. The deterministic cascade is ~89% of an app; language models are used only for the integration glue.

Quick Start

Get up and running in minutes

Installation 
# Install with pip
pip install startd8

# Install with all providers
pip install startd8[all]

# Or install specific providers
pip install startd8[anthropic,openai]
Basic Usage 
from startd8 import AgentFramework, BenchmarkRunner
from startd8.providers import ProviderRegistry

# Initialize framework
framework = AgentFramework()
ProviderRegistry.discover()

# Create a versioned prompt
prompt = framework.create_prompt(
    content="Explain quantum computing in simple terms",
    version="1.0.0",
    tags=["science", "explainer"]
)

# Get providers and create agents
anthropic = ProviderRegistry.get_provider("anthropic")
openai = ProviderRegistry.get_provider("openai")

anthropic.validate_config({})
openai.validate_config({})

agents = [
    anthropic.create_agent("claude-sonnet-4-20250514"),
    openai.create_agent("gpt-4o"),
]

# Run benchmark
runner = BenchmarkRunner(framework)
results = runner.run_benchmark(
    prompt_content=prompt.content,
    agents=agents,
    benchmark_name="quantum-explainer"
)

# View comparison
print(f"Fastest: {results['comparison']['rankings']['by_speed'][0]['agent']}")
print(f"Total cost: ${sum(r['cost_estimate'] for r in results['comparison']['responses']):.4f}")

Provider Pattern

Providers are registered via entry points. Add custom providers without modifying core code.

[project.entry-points."startd8.providers"]
anthropic = "startd8.providers.anthropic:AnthropicProvider"
openai = "startd8.providers.openai:OpenAIProvider"

[project.entry-points."startd8.languages"]
python = "startd8.languages.python:PythonLanguageProfile"
prisma = "startd8.languages.prisma:PrismaLanguageProfile"

Cost Tracking & Budget Management

Never exceed your LLM budget again

Cost Tracking Example 
from startd8.costs import CostTracker, BudgetManager, CostAnalytics

# Initialize cost tracking
tracker = CostTracker()
budget = BudgetManager(tracker)

# Set a monthly budget
budget.set_budget(
    name="monthly-limit",
    amount=100.00,
    period="monthly",
    alert_threshold=0.8  # Alert at 80%
)

# Track costs automatically with agents
agent = anthropic.create_agent(
    "claude-sonnet-4-20250514",
    cost_tracker=tracker
)

# Get analytics and optimization recommendations
analytics = CostAnalytics(tracker)
summary = analytics.get_summary()
recommendations = analytics.get_optimizations()

Session Tracking & Observability

OpenTelemetry traces/metrics/logs to a Loki/Grafana stack (with Mimir/Tempo), plus per-provider cost tracking

Session Tracking Example 
from startd8 import AgentFramework

# Enable session tracking + OpenTelemetry export
framework = AgentFramework(
    enable_session_tracking=True,
    enable_otel=True  # traces/metrics/logs via OTLP → Loki/Grafana
)

# Start a tracked session
session_id = framework.start_session(
    agent_name="claude",
    model="claude-sonnet-4-20250514"
)

# Get session summary
summary = framework.get_session_summary()
print(f"Active sessions: {summary['active_sessions']}")
print(f"Context usage: {summary['average_context_usage']:.1%}")
print(f"Total cost: ${summary['total_cost']:.4f}")

# End session when done
framework.end_session(session_id)
