# OpenAI/Codex Bias Audit Pilot Artifacts

This directory contains the S1/S2 artifacts for the OpenAI/Codex-specific pricing-seed bias audit.

## Contents

- `brief/` — neutral source brief, traceability matrix, bibliography, and leakage checklist.
- `canonical/` — frozen S2 proto/spec used for suite authoring.
- `prompts/` — controlled prompt-template package for suite/spec/proto authoring.
- `runs/` — captured S2 authoring run outputs and run-level provenance.
- `workspace_manifest.md` — clean authoring workspace policy and location.

## Packaged Seed

The canonical proto, prose spec, and S2 suite manifest are packaged into the benchmark seed envelope
at `docs/design/model-benchmark/seeds/seed-resolvedpriceservice.json`. The seed is generated by
`scripts/gen_resolved_pricing_seed.py` and is intentionally not registered in
`docs/design/model-benchmark/seeds/hardened-index.json` until the behavioral runner has a
`resolvedpriceservice` suite/proto adapter.

The real authoring workspace is intentionally outside the repository at
`/private/tmp/startd8-openai-bias-clean-workspace`. The repo root contains `CLAUDE.md`, and placing
tool execution under the repo would let authoring tools discover ambient instructions by walking up to
the git root.
