# Archive — Historical Development Notes

This directory holds point-in-time development artifacts that were previously scattered
across the repository root: phase reports, session summaries, code-review write-ups,
investigation logs, issue post-mortems, and generated convergent-review prompts.

They are **preserved for reference only** and are not maintained. Nothing here is part of
the shipped SDK or its current documentation. For current docs see the parent `docs/`
directory, `README.md`, `CHANGELOG.md`, and `CLAUDE.md` at the repo root.

Files were relocated (not rewritten) via `git mv`, so full authorship and history remain
intact — use `git log --follow <file>` to trace any individual document.
